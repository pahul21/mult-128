# Advanced Website - Elara Café

This project was created as part of the Advanced Website assignment. The goal of this project was to design and develop a multi-page responsive website using modern web development techniques.

## Pages Included
- Home
- About
- Menu
- Gallery
- Events
- Contact

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Sass (SCSS)
- jQuery
- Animate.css

## Features
- Fully responsive design using Bootstrap
- Clean and simple user interface
- Navigation bar for easy access to all pages
- Use of cards and grid layout
- Image gallery section
- Contact form layout
- Sass variables for styling

## Version Control
Git was used to track changes and manage the project. Regular commits were made during development and the project was pushed to GitHub.
