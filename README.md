# Elara Café Website

Advanced Website assignment for MULT-128.

## Project Overview
Elara Café is a fictional café brand website that i created as a multi-page responsive website using content and branding from my social media/brand project.

## Pages
- Home
- About
- Menu
- Gallery
- Contact
- Reservations

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery
- Animate.css
- Google Fonts
- Git and GitHub

## Features
- Responsive layout
- Bootstrap navigation
- Menu filter using jQuery
- Contact form
- Reservation form
- Image gallery
- CSS animations using Animate.css
